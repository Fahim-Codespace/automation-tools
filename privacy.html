Privacy Policy

Last updated: February 2026

This Privacy Policy explains how the Aitech Venture Automation Tool (the “Application”) accesses and uses information when connected to Google services.

1. Purpose of the Application

The Application is an internal automation tool built using the n8n automation platform. It is used to automate workflows such as reading, writing, filtering, and processing data from Google services like Google Sheets, Google Drive, and Gmail.

2. Data Access

The Application only accesses Google account data that is explicitly authorized by the user through the Google OAuth consent screen.

  • Access is limited to the scopes approved by the user
  • Data is accessed only during workflow execution
  • No background or hidden data access occurs

3. Data Usage

Accessed data is used solely to perform user-configured automation workflows. The data is not used for advertising, profiling, or analytics purposes.

4. Data Storage and Sharing

  • No Google user data is sold, rented, or shared with third parties
  • No Google user data is stored permanently outside Google services
  • Temporary data may be processed in memory only during workflow execution

5. User Control

Users have full control over their data and may revoke access at any time through their Google Account security settings.

6. Security

Reasonable security measures are followed to ensure that data access is limited to authorized automation workflows only.

7. Changes to This Policy

This Privacy Policy may be updated from time to time. Continued use of the Application indicates acceptance of the updated policy.
